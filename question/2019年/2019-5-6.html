# 1.解答: 如何用js加载js脚本？

/**
 * 方式一
 */
function loadScript(src: string, async: boolean = true) {
  return new Promise((resolve, reject) => {
    const script = document.createElement('script');
    async && (script.async = true);
    script.onload = resolve;
    script.onerror = reject;
    script.src = src;
    document.body.appendChild(script);
  });
}
// 注意：使用appendChild，不要使用append，ios9不支持append。
/**
 * 方式二
 */
document.writeln('<script src="https://appx/web-view.min.js"' + '>' + '<' + '/' + 'script>');
注意：
1. writeln或write，必须在onload之前执行，不然会覆盖页面内容
2. 如果写在script标签内，必须拆开写</script>，不然会被解析成闭合标签
3. 标签会被插入到执行此script的标签之后，不能指定插入的位置

# Tips: document.writeln/document.write方法触发的一些浏览器机制

浏览器处理页面时首先打开输入流(open()),
开始渲染页面，
页面渲染结束后关闭流(close())。
document.write()在页面加载完毕后（onload）执行，
会重新打开输入流(open())，
输入流被2次打开时，
就会清除之前的页面内容，
所以有的时候document.write()会覆盖页面内容，
有的时候会追加内容。
所以页面只要在onlaod之前使用doucment.write()，
就不会替换掉当前页面的内容.

# 2.解答: 给元素设置position属性,不设置width. 元素默认width是多少？

# 知识点回顾：

  1. 元素为block时，不给元素设置position属性，那么元素默认的宽度则等于父元素的宽度
  2. position属性的值有：unset、inherit、initial、absolute、relative、fixed、 static、sticky
  3. 给一个block元素设置position属性,并且值为：absolute。在不考虑left、right的情况下，该元素的宽度则等于其最近的祖宗元素有position属性的宽度
  4. 给一个inherit元素设置了position属性,并且值为：absolute。在不考虑left、right的情况下, 则元素的宽度等于其最近的祖宗元素有position属性的宽度

# 解：

如果只考虑父子元素的position属性的值为:relative|absolute|fixed，那么该问题的解有以下几种：

点击查看效果：当该元素为块级元素时 (opens new window)

  1. 元素设置position属性并且值为absolute、relative，不设置left或者right的值。则该元素的宽度是等于其最近的祖宗元素有position属性的宽度
  2. 元素设置position属性并且值为fixed，不设置left或者right的值。则该元素的宽度是等于window窗口的宽度
  3. 元素设置position属性并且值为fixed，设置left值为10%，那么默认宽度为：窗口的宽度 - 10%
  4. 元素设置position属性并且值为fixed，设置left值为10%，right值为20%，那么默认宽度为：窗口的宽度 - 10% - 20%
  5. 元素设置position属性并且值为absolute, 设置left、right的值为 x。那么元素的默认宽度为：其最近的祖宗元素有position属性的宽度 - x * 2
  6. 元素设置position属性并且值为relative, 设置left或者right的值为 x。那么元素的默认宽度为：其最近的祖宗元素有position属性的宽度

点击查看效果：当该元素为行内元素时 (opens new window)

  1. 元素设置position属性，不设置left或者right的值。则元素的默认宽度为其内部文字撑起来的宽度
  2. 元素设置position属性并且值为absolute, 设置left、right的值为x。那么元素的默认宽度：其最近的祖宗元素有position属性的宽度 - x * 2
  3. 元素设置position属性并且值为fixed, 设置left、right的值为x。那么元素的默认宽度：窗口的宽度 - x * 2
  4. 元素设置position属性并且值为relative, 设置left、right的值为 x。那么元素的默认宽度为：其内部文字撑起来的宽度

PS: 如果考虑父子元素position属性的值再加上: static|sticky|initial|inherit|unset请自行demo验证总结
